## 四、EF Core 继承层次对表字段的影响

### FullAuditedAggregateRoot<Guid>

AppTargetDatabases 和 AppDbTables 使用此基类，包含以下审计字段:

- **创建审计**: `CreationTime`, `CreatorId`
- **修改审计**: `LastModificationTime`, `LastModifierId`
- **软删除**: `IsDeleted`, `DeletionTime`, `DeleterId`
- **并发控制**: `ConcurrencyStamp`
- **扩展**: `ExtraProperties`

### FullAuditedEntity<Guid>

AppDbFields 使用此基类，包含以下审计字段:

- **创建审计**: `CreationTime`, `CreatorId`
- **修改审计**: `LastModificationTime`, `LastModifierId`
- **软删除**: `IsDeleted`, `DeletionTime`, `DeleterId`

### CreationAuditedEntity<Guid>

AppSqlExecutionLogs 使用此基类，仅包含:

- **创建审计**: `CreationTime`, `CreatorId`

## 五、ExecutionStatus 枚举说明

| 值 | 名称       | 说明                          |
| - | -------- | --------------------------- |
| 0 | Pending  | 待执行 - 字段已定义但 SQL 尚未在目标数据库执行 |
| 1 | Executed | 已执行 - SQL 已在目标数据库执行成功       |
| 2 | Failed   | 执行失败 - SQL 执行过程中出错          |
